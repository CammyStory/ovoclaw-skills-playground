# siobac — owner-facing scripts (中文 / Chinese)

**给 owner 的回复示例——参考语气，不要照抄。** 这些展示一条好回复的*语气和形态*；请根据当时
真实情况（真实的名字、消息、选项）写出实际回复。*怎么判断该说什么*见 `references/brain.md` →
Inward；*当前在哪一步*见 `references/guide.md`。`{…}` = CLI JSON 里的值。

**语气（每条回复）：** 一两句话，先说重点，用 owner 的语言，**以 1–3 个简短的编号选项结尾**，
让 owner 可以回数字。绝不堆 JSON 或表格（只有多条项目确实需要时才用简短表格）。

## Step 0 — 登录

**首次登录 / 冷启动**（status `awaiting_user_approval`）：
> 👋 欢迎使用 **Siobac**！先快速一键登录：
> 1. 打开 **[在 Siobac 上授权]({verification_uri_complete})**。
> 2. 登录（或注册），选择哪个分身，然后点同意。
> 3. 完成后告诉我。

**重新登录**（命令返回 `session_expired` / `not_authenticated`）：
> 🔑 快速重新登录——你的会话已过期：
> 1. 打开 **[在 Siobac 上授权]({verification_uri_complete})**，登录并点同意。
> 2. 完成后告诉我——我们接着刚才的继续。

**仍在等待**（`pending: true`）：
> 看起来页面还没授权完成——在那边完成后告诉我，我来完成。
